Factorial (n!)

The factorial of n is denoted by n! and calculated by the product of integer numbers from 1 to n.
Examples:
1! = 1
2! = 1×2 = 2
3! = 1×2×3 = 6
4! = 1×2×3×4 = 24
5! = 1×2×3×4×5 = 120
Recursive factorial formula
n! = n×(n-1)!
Example: 5! = 5×(5-1)! = 5×4! = 5×24 = 120

JS example code

function factorial(n) {
 let fact = 1;
 for (let i = 1; i < n; i++) {
  fact += fact * i;
 }
 return fact;
}

Fibonacci Numbers & Sequence

Fibonacci sequence is a sequence of numbers, where each number is the sum of the 2 previous numbers, except the first two numbers that are 0 and 1.
Examples:
F0 = 0
F1 = 1
F2 = F1 + F0 = 1 + 0 = 1
F3 = F2 + F1 = 1 + 1 = 2
F4 = F3 + F2 = 2 + 1 = 3
F5 = F4 + F3 = 3 + 2 = 5
...

function fibonnaci(n) {
 let fn = n;
 let f0 = 0;
 let f1 = 1;
 if(n < 1) return n;
 for (let i = 2; i <= n; i++) {
  fn = F0 + F1;
  F0 = F1;
  F1 = fn;
 }
 return F1;
}
